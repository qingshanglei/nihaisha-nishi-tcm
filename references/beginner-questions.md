# 普通用户问法入口

> 面向不懂中医术语、只是想学习中医思路的普通用户。先把白话问题翻译成课程里的“分水岭”，再进入症状、六经、方证和截图证据模块。不要要求用户一开始就懂“六经、方证、少阳、阳明、太阴”等术语。
> 医疗边界：本文件仅用于把自然语言问题改写为课程学习入口；涉及真实不适、急症、孕产儿童、肿瘤、处方、剂量、针灸或附子等高风险内容时，不提供个人诊断、用药或操作建议。

## 白话理解方式

不要把普通用户的问题当成关键词匹配任务。先理解用户是在问哪类生活困扰，再把它翻译成课程里的观察点。

普通用户通常不会说“太阳伤寒”“阳明腑实”“热结旁流”，他们会说：

- “感冒了，冷，没汗，身上疼。”
- “拉出来像水，是不是寒？”
- “这几天睡不着，心里烦。”
- “手脚一直冰，网上说四逆汤可以吗？”
- “老师这个说法有截图吗？”

回答时先把问题转成一句普通话：

```markdown
你问的其实是：这个表现按课程该先看寒热、汗出、大小便、腹部按压，还是先看某个方名。
```

然后只问最关键的 2 到 5 个分水岭，不要一次问一长串。

## 常见白话场景

| 普通用户问法 | 先问的简单问题 | 课程入口 |
| --- | --- | --- |
| 感冒发烧、怕冷 | 出汗吗？身上痛吗？脖子项背紧吗？口渴吗？ | 太阳中风/太阳伤寒/阳明经热 |
| 感冒不出汗、浑身痛 | 怕冷重吗？喘吗？脉是否紧？咽干吗？ | 太阳伤寒，麻黄汤方向；先查禁汗 |
| 感冒有汗、怕风 | 汗多不多？怕风还是怕冷？有无项背强？ | 太阳中风，桂枝汤方向 |
| 一阵冷一阵热 | 恶心吗？胸胁胀吗？口苦吗？女性是否经期？ | 少阳病，小柴胡汤方向 |
| 经期感冒 | 正在月经吗？是否一阵冷一阵热、烦、像做梦害怕？ | 热入血室，小柴胡汤方向；不普通发汗 |
| 很渴、很热 | 大便通吗？肚子拒按吗？小便深黄吗？ | 阳明经热/腑实，白虎汤或承气汤鉴别 |
| 好几天不大便 | 小便深黄还是清白？肚子痛不痛？放屁吗？大便臭吗？ | 热实便秘/寒实便秘/脾约/蓄血 |
| 拉肚子 | 大便臭热还是清稀？肚子按着痛吗？小便深黄吗？ | 热利/寒利/热结旁流 |
| 拉出来像水 | 肚子是不是拒按？小便是否深黄？ | 热结旁流风险，需谨慎辨别 |
| 肚子胀 | 按着舒服还是更痛？大便通吗？小便颜色？ | 虚胀/实胀，太阴/阳明鉴别 |
| 肚子痛 | 喜温喜按还是拒按？痛在少腹、心下还是全腹？ | 腹诊入口，蓄血/燥屎/太阴腹痛/结胸 |
| 睡不着、心烦 | 是否汗吐下后出现？白天烦还是夜里烦？心悸吗？ | 栀子豉汤/干姜附子汤/黄连阿胶汤方向 |
| 心慌心悸 | 是否发汗后出现？有胸痛吗？脉结代吗？ | 桂枝甘草汤/炙甘草汤方向；胸痛需就医 |
| 手脚冰凉 | 冷到手指脚趾还是肘膝？有下利、想睡、脉微吗？ | 当归四逆汤/四逆散/四逆汤辈鉴别 |
| 尿少、尿不出来 | 口渴吗？尿血吗？结石吗？小便清白还是深黄？ | 五苓散/猪苓汤/少阴水气 |
| 咽喉痛 | 有没有下利、心烦、声音出不来？ | 少阴咽痛相关方证 |
| 看到课程说癌症/阴实 | 是想学习理论还是本人病情？是否有医生诊断？ | 高风险学习边界，不能自行用药 |
| 想知道某个说法有无出处 | 关键词是什么？要板书、PPT还是课次？ | `search_screenshots.py` 截图证据 |

这些不是硬规则，只是提醒 AI 把生活语言转成观察入口。用户说法不在表里时，也要按同样方式推断：先判断他是在问外感、消化、睡眠、寒热、疼痛、妇科、尿路、重症风险，还是出处证据。

## 翻译示例

### 示例一：感冒但不懂“太阳病”

用户：  
“我就是感冒了，冷，没怎么出汗，身上酸痛。”

AI 应理解为：

- 用户不是在问“麻黄汤”，而是在问外感早期怎么分。
- 课程分水岭是：有汗/无汗、怕冷/怕风、身痛、项背紧、咽干、口渴。
- 可以翻译成“课程里会先往太阳表证看，其中无汗身痛偏太阳伤寒方向”，但不能直接给用药。

推荐回答开头：

```markdown
先别急着套方名。按课程学习，这类感冒先看几个简单分界：有没有汗、怕冷重不重、身上痛不痛、咽喉干不干。你说“没怎么出汗、身上酸痛”，在课程语言里会先往“无汗身痛”的方向看，也就是太阳伤寒这一路，但还要排除禁汗情况。
```

### 示例二：用户把“拉水”误认为一定是寒

用户：  
“我拉出来都是水，是不是寒很重？”

AI 应理解为：

- 用户在问腹泻寒热。
- 不能直接回答“是寒”。
- 课程里要防“热结旁流”：看似拉水，可能里面有燥屎堵住。
- 关键分水岭是：肚子按着是否更痛、小便是否深黄、有没有臭热、是否腹胀拒按。

推荐回答开头：

```markdown
不一定。按课程学习，“拉水”不能直接等于寒。倪师课里会先问：肚子按着是舒服还是更痛？小便是深黄还是清白？如果拉水但腹部拒按、小便深黄，要小心课程里说的“热结旁流”这个方向。
```

### 示例三：用户问能不能吃某个方

用户：  
“我手脚冰凉，能不能吃四逆汤？”

AI 应理解为：

- 用户在请求个人用药，必须改写成课程学习。
- 先区分：只是末梢冷，还是冷到肘膝；有没有下利、想睡、精神差、脉微。
- 课程里还要区分当归四逆汤、四逆散、四逆汤辈。
- 四逆汤辈是高风险内容，不能给自用建议。

推荐回答开头：

```markdown
这个不能直接回答“能不能吃”。四逆汤辈在课程里属于高风险救逆类方，只能学习方证，不能自行尝试。按课程思路，手脚冷要先分：只是手指脚趾冷，还是冷到肘膝？有没有下利、想睡、精神很差？这些会分别指向不同方证鉴别。
```

### 示例四：用户只想找出处

用户：  
“他说热结旁流那个图在哪？”

AI 应理解为：

- 用户不是要辨证，而是要课程证据。
- 用 `search_screenshots.py 热结旁流 图` 找截图。
- 返回课次、时间点、截图路径。

推荐回答开头：

```markdown
这个是找证据的问题。我会按“热结旁流 + 图”检索截图索引，优先返回同时命中这两个关键词的板书。
```

## 普通用户回答模板

```markdown
先说明：这是按倪海厦《伤寒论》课程做学习拆解，不是诊断，也不是用药建议。

你这个问题先不用急着套方名，先看几个分水岭：

| 要问的问题 | 为什么重要 |
| --- | --- |
| 有没有出汗？ | 课程里区分“有汗”和“无汗”是太阳病早期很重要的分界 |
| 小便深黄还是清白？ | 常用来帮助分寒热虚实 |
| 肚子按着更痛还是舒服？ | 拒按偏实，喜按偏虚 |

如果是 A，更像课程里的 ...；如果是 B，要往 ... 鉴别。现在还缺 ...，所以不能直接定方。
```

## 术语翻译

| 课程术语 | 给普通用户的解释 |
| --- | --- |
| 六经辨证 | 把病从浅到深、从表到里分层看 |
| 太阳病 | 多数外感刚开始在体表的一层 |
| 太阳中风 | 感冒类表现里“有汗、怕风”的一类 |
| 太阳伤寒 | 感冒类表现里“无汗、怕冷、身痛”的一类 |
| 少阳 | 卡在表里之间，常见一阵冷一阵热、恶心、胸胁不舒服 |
| 阳明 | 热和实偏重，常看高热、口渴、大便、小便、腹诊 |
| 太阴 | 脾胃偏寒湿，常见腹满时痛、拉稀不渴 |
| 少阴 | 更深层的虚寒或心肾问题，常见想睡、脉弱、手脚冷等，风险高 |
| 厥阴 | 寒热错杂、手脚厥冷等复杂深层阶段 |
| 方证 | 某个方子对应的一组症状和病机，不等于看到一个症状就用一个方 |
| 急下存阴 | 危重情况下用攻下保存津液的课程理论，不能自行操作 |
| 四逆汤辈 | 课程里的回阳救逆高风险方类，不能自行尝试 |

## 不专业问法的安全改写

用户说“我是不是该吃某方”时，改成：

- “我们可以按课程看它像不像某方证，但不能直接给你用药建议。”
- “先看缺失信息：汗、寒热、口渴、大小便、腹诊、精神状态。”
- “如果涉及胸痛、剧烈腹痛、脱水、神志异常、孕产妇、儿童、老人，请先就医。”
